Bootstrap 4

  1. Bootstrap 4 vs Bootstrap 3
  2. Bootstrap 3 doesn't use the .nav-item or .nav-link classes.
  3. Bootstrap 4 requires .nav-item to be applied to any <li> elements.
  4. Bootstrap 4 requires .nav-link to be applied to the <a> element.
  5. Bootstrap 3 requires any .active class to be applied to the <li> element.
  6. Bootstrap 4 requires any .active class to be applied to the <a> element.
